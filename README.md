# Page to EPUB

Chrome MV3 插件，用于把 blog / 新闻类网页正文提取出来并导出成 EPUB。

## 功能

- 右键当前页面直接导出 EPUB
- 右键链接，在后台打开链接并导出 EPUB
- 基于 Mozilla Readability 抽取正文
- 默认去除广告、导航、侧栏等非正文内容
- 保留段落、标题、列表、引用、代码块、表格、图片、基础文字样式
- 默认保存到浏览器下载目录下的 `EPUB/`

## 开发

```bash
npm install
npm run build
```

构建产物会输出到 `dist/`。

## 安装

1. 打开 `chrome://extensions`
2. 开启右上角“开发者模式”
3. 点击“加载已解压的扩展程序”
4. 选择本项目下的 `dist/` 目录

## 使用

- 在文章页面空白处右键，选择“转为 EPUB”
- 或点击插件图标，在弹窗里点击“转为 EPUB”
- 下载文件默认会出现在浏览器下载目录下的 `EPUB/`

## 限制

- 仅支持 `http` / `https` 页面
- 登录态很强、图片防盗链严格、纯前端流式渲染页面，可能出现正文或图片提取不完整
- 如果 Chrome 设置了“下载前询问每个文件的保存位置”，浏览器仍可能弹出保存对话框
